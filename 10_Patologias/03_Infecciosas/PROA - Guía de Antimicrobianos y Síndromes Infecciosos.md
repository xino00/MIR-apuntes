# PROA - Guía de Antimicrobianos y Síndromes Infecciosos

**Fuente:** Protocolo Diagnóstico y Tratamiento de Principales Síndromes Infecciosos en Adultos de Manejo Ambulatorio (FJD, 2023) y Guía de Manejo de Antibióticos (FJD, 2025).

Esta guía resume las indicaciones para el manejo ambulatorio de los síndromes infecciosos más prevalentes y presenta una tabla de dosificación y ajuste de los principales antimicrobianos.

---

## 🦠 1. Síndromes Infecciosos (Manejo Ambulatorio)

Para consultar las indicaciones empíricas de primera línea, alternativas por alergias, y los criterios de derivación hospitalaria según el origen del foco infeccioso, accede a las siguientes guías específicas extraídas del protocolo:

- 🫁 **[[PROA - Infecciones Respiratorias]]**: Faringitis, Otitis, Sinusitis, Bronquitis, EPOC, NAC.
- 🩸 **[[PROA - Infecciones de Transmisión Sexual (ITS)]]**: Uretritis, Cervicitis, EIP, Sífilis, Herpes Genital.
- 💧 **[[PROA - Infecciones del Tracto Urinario (ITU)]]**: Cistitis, Pielonefritis, Prostatitis, Manejo de Sondas/BLEE.
- 🩹 **[[PROA - Infecciones de Piel y Partes Blandas]]**: Impétigo, Celulitis, Abscesos, Mordeduras, Exposiciones Acuáticas.
- 🚽 **[[PROA - Infecciones Gastrointestinales]]**: Diarrea Inflamatoria, Diarrea del Viajero, *Clostridioides difficile*.

## 💊 2. Guía de Dosificación de Antimicrobianos (Uso Frecuente)

Para consultar las pautas estándar, los ajustes avanzados según el FG (< 30 ml/min), consideraciones en Terapias de Reemplazo Renal, insuficiencia hepática y particularidades sobre perfusiones extendidas o medición de niveles séricos, accede a las guías detalladas por familias:

- 🛡️ **[[PROA - Betalactámicos y Cefalosporinas]]**: Penicilina G/V, [[Ampicilina]], [[Amoxicilina]], Amox/Clavulánico, [[Cloxacilina]], Pip/Tazo, [[Cefazolina]], [[Cefuroxima]], [[Cefotaxima]], [[Ceftriaxona]], [[Cefixima]], [[Cefditoreno]], [[Ceftazidima]], Ceftazidima/Avibactam, Ceftolozano/Tazobactam, [[Ceftarolina]], Cefiderocol, [[Cefepime]], [[Aztreonam]], [[Ertapenem]], Imipenem/Cilastatina, [[Meropenem]], Meropenem/Vaborbactam, [[Rifampicina]].
- 🎯 **[[PROA - Quinolonas y Macrólidos]]**: [[Ciprofloxacino]], [[Levofloxacino]], [[Moxifloxacino]], [[Azitromicina]], [[Claritromicina]], [[Clindamicina]], [[Doxiciclina]], [[Linezolid]].
- ⚖️ **[[PROA - Otros Antimicrobianos]]**: [[Tobramicina]], [[Amikacina]], [[Gentamicina]], [[Vancomicina]], [[Teicoplanina]], [[Daptomicina]], [[Dalbavancina]], [[Tigeciclina]], Fidaxomicina, Colistina, [[Fosfomicina]], [[Cotrimoxazol]], [[Metronidazol]].

> [!important] Ajuste Renal General
> El Filtrado Glomerular (FG) marca el ritmo de administración. Las **dosis de carga** (ej. Vancomicina o Aminoglucósidos en situaciones críticas) generalmente **NO** se modifican en insuficiencia renal per se, solo se ajustan las posteriores de mantenimiento.

## ⚖️ 3. Ajuste por Peso Corporal

Ciertos antimicrobianos requieren calcular la dosis según el **peso corporal ideal o ajustado**, especialmente en pacientes obesos o con composición corporal atípica.

| Fórmula | Cálculo | Cuando usar |
| :--- | :--- | :--- |
| **Peso Corporal Ideal (PCI)** | Hombre: 50 + 0.75 × (talla cm − 152.4) · Mujer: 45.5 + 0.67 × (talla cm − 152.4) | Punto de partida en cualquier ajuste |
| **Peso Ideal Ajustado (PIA)** | PCI + C × (PCT − PCI). C = 0.4 para la mayoría; C = 0.45 para [[Ciprofloxacino]] | Si PCT > 1.2 × PCI (obesidad) |
| **Peso Corporal Total (PCT)** | Peso del paciente en balanza | Ver indicaciones por fármaco |

| Fármaco | Peso para dosis | Nota |
| :--- | :--- | :--- |
| Aminoglucósidos (Amikacina, Gentamicina, Tobramicina) | **PIA** (para 1ª dosis, luego ajustar por niveles) | Usar PIA si PCT > 1.2 × PCI |
| Vancomicina (dosis carga) | **PCT** (máximo 3g) | No ajustar carga por obesidad |
| Vancomicina (mantenimiento) | **PIA** si obeso | Titular por niveles |
| Daptomicina | **PIA** | Datos limitados en super-obesidad |
| Colistina | **PCT** | Requiere cálculo específico de CBA |
| Betalactámicos, Quinolonas | **PCT** salvo indicación específica | Generalmente no requieren ajuste |

## 🔄 4. Biodisponibilidad Oral — Guía de Paso IV → VO

El paso de antibiótico intravenoso a oral puede realizarse de forma segura cuando el paciente tolera vía oral, no hay malabsorción y la biodisponibilidad del fármaco es alta (> 80%). No es necesario esperar a completar 48-72h IV si hay respuesta clínica y el fármaco oral tiene biodisponibilidad equivalente.

| Antibiótico | Biodisponibilidad oral | Paso IV → VO seguro |
| :--- | :--- | :--- |
| [[Linezolid]] | ~100% | ✅ Siempre (IV = VO) |
| [[Metronidazol]] | 90-100% | ✅ Siempre |
| [[Levofloxacino]] | ~99% | ✅ Siempre |
| [[Rifampicina]] | ~90-95% | ✅ Siempre |
| [[Doxiciclina]] | ~93% | ✅ Siempre |
| [[Cotrimoxazol]] | ~90-95% | ✅ Siempre |
| [[Clindamicina]] | ~90% | ✅ Siempre |
| [[Moxifloxacino]] | ~91% | ✅ Siempre |
| [[Ciprofloxacino]] | 70-80% | ✅ Generalmente |
| [[Amoxicilina]] | ~70% | ✅ Generalmente |
| [[Claritromicina]] | ~50% | ⚠️ Con cautela en infecciones graves |
| [[Cefixima]] | 40-50% | ⚠️ Adecuado para ITU/faríngeas, no para sepsis |
| [[Azitromicina]] | ~38% (tisular alta) | ⚠️ Concentración tisular muy alta (útil en atípicos) |
| Amox/Clavulánico | ~75% (AMX) / ~60% (CLV) | ⚠️ Adecuado para infecciones de baja/media gravedad |
| [[Cefuroxima]] | ~52% (administrar con alimentos) | ⚠️ Adecuado para infecciones leves-moderadas |
| [[Cloxacilina]] | ~43% (administrar en ayunas) | ⚠️ Solo para infecciones ambulatorias |
| Penicilina V | ~60% (administrar en ayunas) | ⚠️ Adecuado para faringitis estreptocócica |
| [[Cefditoreno]] | ~15-20% (aumenta con comida) | ❌ Solo para infecciones ambulatorias leves |
| Eritromicina | ~35-75% | ⚠️ Variable, usar con cautela |
| [[Fosfomicina]] trometamol | ~33-53% (administrar en ayunas) | ⚠️ Adecuado para cistitis (acción local) |
| Betalactámicos IV (Ceftriaxona, Meropenem…) | 0% | ❌ No disponibles vía oral |
